That's a lot of features
When you first open After Effects, don’t be overwhelmed with the interface, make sure to spend some time getting familiar with the interface's various panels, menus, and features like the Project panel, Composition panel, Timeline panel, and various other panels for different tasks.
Compositions are the main building blocks in After Effects. They represent individual projects or scenes within a project. Within compositions, you work with layers, which are similar to Photoshop layers and can include images, videos, text, shapes, and effects. Learn how to create compositions and add, organize, and manipulate layers within them.

After Effects tricks
Some of the tips and tricks that helped me use AE easily:
  • Organize your project: Keep your project organized by creating folders in the Project panel to store assets, compositions, and other elements.
  • Using presuppositions: Precomp allows you to group layers together and treat them as a single entity within another composition. This is particularly useful when you want to apply effects, animations, or transformations to multiple layers simultaneously.
  • Master keyframing: Keyframes are essential for creating animations in After Effects. Learn how to set keyframes for different properties and experiment with easing and interpolation to achieve smooth and natural motion.
